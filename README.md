# Protobuf-Hex-Inspector

A lightweight, client-side web tool to decode **schema-less Protobuf HEX payloads** into structured JSON.

Designed for debugging and reverse-engineering when the original `.proto` schema is unavailable.  
No backend. No uploads. No tracking.

## Features
- Decode Protobuf HEX payloads
- Schema-less decoding (numeric field keys)
- Supports nested and repeated fields
- Automatic UTF-8 string detection
- Clean JSON output
- Fully client-side

## Usage
1. Paste Protobuf HEX payload
2. Click **Decode**
3. View decoded JSON output
